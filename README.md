# Senior Coffee Scramble

A simple webapp to allow seniors of the Columbia University class of 2014
to invite each other to grab coffee/food/whatever and get to know each other
better.

## Prerequisites

You will need [Leiningen][1] 1.7.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein run server
